🎯 HOW IT WORKS

Workflow 1: Admin Upload
  Admin → /dashboard/gallery/add/ → Upload image → Save → Displays on /gallery/

Workflow 2: Auto-Sync Job Photos
  Employee uploads job photo → Signal triggered → GalleryItem auto-created
  → Image appears on /gallery/ immediately

Workflow 3: Public Viewing
  User → /gallery/ → Filters → View full-size → Staff can delete

═══════════════════════════════════════════════════════════════════════════════

🌐 URLS

Public:
  /gallery/ - View all images (with filters)
  /gallery/?service=Office%20Cleaning - Filtered view
  /gallery/?source=job_photo - Filtered by source
  /gallery/<id>/delete/ - AJAX delete (staff only)

Dashboard:
  /dashboard/gallery/ - Gallery manager
  /dashboard/gallery/add/ - Add image
  /dashboard/gallery/<id>/edit/ - Edit image
  /dashboard/gallery/<id>/delete/ - Delete image

Admin:
  /admin/gallery/galleryitem/ - Gallery admin
  /admin/gallery/galleryitem/add/ - Add image
  /admin/gallery/galleryitem/<id>/change/ - Edit image
  /admin/gallery/galleryitem/<id>/delete/ - Delete image
